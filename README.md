# 📌 Task Management System 

A simple and interactive **Task Management System** built using **HTML, CSS, and JavaScript**.
This project allows users to efficiently manage daily tasks with features like adding, editing, deleting, completing, filtering tasks, and saving data using local storage.

## 🚀 Features

* ➕ Add new tasks with due date and priority
* ✏️ Edit existing tasks
* ❌ Delete tasks
* ✅ Mark tasks as completed
* 🔢 Task counter (total tasks)
* 🔍 Filter tasks (All / Completed / Pending)
* 💾 Data persistence using Local Storage

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Git & GitHub (for version control and branching)

## 🌿 Git Branching Strategy

This project follows a **feature-based branching approach**:

* `main` → Final integrated project
* `add-task-feature` → Add task functionality
* `edit-task-feature` → Edit task functionality
* `delete-task-feature` → Delete task functionality
* `complete-task-feature` → Complete task functionality
* `task-counter-feature` → Task counting logic
* `filter-task-feature` → Filtering tasks
* `storage-feature` → Local storage handling

Each feature was developed independently and later merged into the `main` branch.

## ▶️ How to Run the Project

### Method 1:

* Open the project folder
* Double-click `index.html`

### Method 2 (Recommended):

* Open in VS Code
* Install **Live Server extension**
* Right-click `index.html` → **Open with Live Server**

## 🧪 How to Use

1. Enter a task in the input field
2. Select due date and priority
3. Click **Add Task**
4. Use buttons to:

   * Edit ✏️
   * Delete ❌
   * Mark Complete ✅
5. Use filters to view tasks
6. Refresh page → tasks remain saved
